# 첨부 시안 일치 UI 및 GitHub Pages 경로 보정

## 기준

- 사용자 첨부 `codex-clipboard-e7f85d10-0d4e-4f7f-8baa-fc3a5f5d5110.png`을 화면의 직접 기준으로 삼는다.
- 첫 화면은 로고·보조 링크, 넓고 얕은 말풍선 미리보기, 가로 설정막대, 예시가 채워진 입력 영역과 빈도표로 구성한다.

## 필수 보정

1. 정적 내보내기 HTML의 `/assets/` 절대 경로를 `./assets/` 상대 경로로 변환해 GitHub Pages 프로젝트 하위 경로에서 CSS·JS를 불러온다.
2. 미리보기는 짙은 남색 외곽선, 종이 질감 배경, 오른쪽 아래 PNG 버튼의 얕은 넓은 비율을 사용한다.
3. 설정은 마스크·색상·글꼴·단어 개수를 한 줄 도구막대에 배치하고, 선택 상태는 파란 테두리와 체크로 보인다.
4. 초기 화면에는 수업 예시 문장과 해당 빈도표를 보여주되, 입력 원문은 브라우저 메모리에만 둔다.
5. 반응형에서는 도구막대와 두 패널을 세로로 쌓는다.

## 검증

- 자산 경로 200 응답, 테스트·정적 빌드, 데스크톱·모바일 브라우저 캡처, GitHub Pages 실제 URL을 확인한다.
