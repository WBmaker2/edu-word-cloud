# 말풍선·하트·책 마스크 형태 다듬기 계획

## 문제

- 말풍선과 하트 외곽선은 내부 판정 함수를 일정 간격으로 추적해 그린다. 이 방식은 곡선이 미세하게 꺾여 보인다.
- 책 마스크는 1200×500 미리보기에서 가로:세로가 약 2.7:1이라, 펼친 책보다 납작한 띠처럼 보인다.

## 구현 범위

1. 말풍선과 하트를 직접 정의한 베지어 곡선 경로로 그려 외곽선을 매끈하게 한다.
2. 책을 세로 비율이 살아 있는 열린 책 실루엣으로 바꾸고, 중앙 제본선을 더한다.
3. 단어 배치용 내부 판정과 마스크 경계를 새 비율에 맞춰 보수적으로 조정한다.
4. 곡선 경로와 책 비율의 회귀 테스트를 추가하고, 업데이트 내역에 개선 기록을 남긴다.

## 검증

- `npm test`, `npm run build:static`, `git diff --check`를 실행한다.
- 브라우저에서 말풍선·하트·책을 각각 선택해 매끈한 외곽선, 책 제본선, 단어 배치 상태를 확인한다.
