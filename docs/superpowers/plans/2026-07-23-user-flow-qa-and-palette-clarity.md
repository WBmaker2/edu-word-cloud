# 2026-07-23 실사용 흐름 점검 및 색상 조작 명확화

## 점검에서 찾은 개선점

1. 색상 화살표의 첫 클릭은 맨 왼쪽 팔레트부터 고르는 특별한 규칙이지만, 화면에는 색상 견본과 화살표만 보여 처음 사용하는 교사가 동작을 예상하기 어렵다.
2. 2열로 바뀌는 중간 화면 폭에서 설정 그룹의 테두리를 DOM 자식 순서로 지정하고 있어, 제목 요소를 포함한 순서 변화에 따라 실제 2×2 배치와 경계가 맞지 않을 수 있다.
3. 색상 견본과 화살표가 조밀해 마우스·터치 조작 시 누르기 여유가 작다.

## 개선 범위

- 색상 그룹에 현재 선택한 팔레트명과 첫 화살표의 동작을 짧게 보이게 한다.
- 상태 문구를 실시간 안내 영역으로 제공해 색만으로 선택 결과를 판단하지 않도록 한다.
- 색상 견본과 이전·다음 버튼의 누르기 영역을 조금 키운다.
- 중간 폭에서는 `fieldset` 기준으로 2×2 격자의 오른쪽·아래 테두리를 정확히 배치한다.
- 단위 테스트와 정적 빌드, 데스크톱·중간 폭·모바일 화면 폭 점검으로 확인한다.

## 제외 범위

- 단어 분석 방식, 개인정보 처리 방식, 미리보기 생성 방식은 변경하지 않는다.
