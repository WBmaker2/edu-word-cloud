# 색상 팔레트 확장·화살표 비교 기능 계획

## 목표

- 색상 선택지를 기존 4개에서 6개로 늘린다.
- 색상 견본 오른쪽에 이전·다음 화살표를 둬 팔레트를 한 칸씩 비교한다.
- 색상을 직접 누르지 않아도, 화살표를 처음 누르면 가장 왼쪽 팔레트부터 선택한다.

## 구현 범위

1. `cloud-options.mjs`에 차분한 보라와 봄빛 파스텔 팔레트를 추가한다.
2. 팔레트 이동 규칙을 별도 모듈로 만들고, 첫 화살표 조작·양끝 제한을 테스트한다.
3. 빠른 설정의 색상 영역에 접근 가능한 SVG 이전·다음 버튼을 추가한다.
4. 선택 상태·미리보기 갱신·브라우저 저장은 기존 `paletteId` 흐름을 그대로 사용한다.
5. 업데이트 내역에 2026-07-23 개선 기록을 남긴다.

## 검증

- `npm test`, `npm run build:static`, 소스 린트를 실행한다.
- 데스크톱과 390px 모바일에서 화살표 첫 클릭, 이전·다음 한 칸 이동, 팔레트 직접 선택, 가로 넘침 여부를 확인한다.
