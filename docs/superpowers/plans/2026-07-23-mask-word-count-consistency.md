# 2026-07-23 마스크별 단어 수 일관성

## 문제

- 현재 배치기는 마스크 안에 자리가 없으면 해당 단어를 생략한다.
- 따라서 같은 `20개` 설정이라도 마스크의 면적·형태에 따라 실제 표시 수가 달라진다.

## 목표

- 분석 결과와 선택한 단어 수로 결정된 동일한 대상 단어 집합을 모든 마스크에 표시한다.
- 마스크가 좁거나 복잡할수록 글자 크기와 여백을 자동으로 줄여서 모두 배치한다.
- 각 단어의 외곽 상자 샘플이 마스크 경계 안에 남도록 확인한다.

## 구현 및 검증

- 배치 실패 시 전체 글자 크기·간격을 단계적으로 축소하며 다시 배치한다.
- 충분히 촘촘한 탐색으로 최종 시도에서도 대상 단어가 생략되지 않도록 한다.
- 20개와 100개 단어를 모든 마스크에 배치하는 테스트를 추가하고, 위치가 마스크 안인지 검증한다.
- 업데이트 내역을 기록한다.
